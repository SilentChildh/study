# 关于Git的操作指令

| 命令                                 | 作用            |
| ------------------------------------ | --------------- |
| git config --global user.name 用户名 | 设置用户签名    |
| git config --global user.email 邮箱  | 设置用户签名    |
|                                      |                 |
| git init                             | 初始化本地库    |
|                                      |                 |
| git status                           | 查看本地库状态  |
|                                      |                 |
| git add <File>                       | 工作区-->暂存区 |
| git -commit -m 'first commit‘        | 暂存区-->本地库 |
| git commit -a -m 'full commit        | 工作区-->本地库 |
